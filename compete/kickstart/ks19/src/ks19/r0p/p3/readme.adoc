power_i 
- generate contiguous subarrays of the parameter array
- sum the i-th exp power of all contiguous subarrays

i-th exp power ofsubarray A~j~, A~j+1~, ..., A~k~ is defined as A~j~ × 1^i^ + A~j+1~ × 2^i^ + A~j+2~ × 3^i^ + ... + A~k~ × (k-j+1)^i^.
